# Environment Setup

## Required Variables
Create a `.env` file in the root directory (never commit this):
```bash
OPENAI_API_KEY=your-openai-api-key-here
ASSISTANT_ID=your-assistant-id-if-precreated
VECTOR_STORE_ID=your-vector-store-id-if-precreated
# Logging / Cron settings if needed
```

## Sample Setup
A `.env.sample` must be placed in the repository root to guide settings.
